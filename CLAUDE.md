- `mcp__gmail__read_emails` - Read emails with filters
- `mcp__gmail__search_emails` - Search with Gmail syntax
- `mcp__gmail__send_email` - Send emails
- `mcp__gmail__create_draft` - Create draft emails
- `mcp__gmail__get_email_thread` - Get full conversation threads
- `mcp__gmail__mark_as_read` - Mark emails as read
- `mcp__gmail__add_labels` - Add labels to emails
- `mcp__gmail__get_labels` - Get all labels

Always use these tools to their full potential to help Edwin manage his inbox efficiently.
